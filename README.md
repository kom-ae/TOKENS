# TOKENS
Автоматизация рутинных процессов по выдаче внутренних подписей предприятия.

Для автоматизации процессов при выдаче УНЭП, выполняет форматирование токенов различных производителей.
Приложение применяется при подготовке токенов к записи закрытого контейнера  и сертификата ключа проверки электронной подписи. При форматировании используются консольные утилиты производителей ключей (Rutoken) для лучшей совместимости или библиотека драйвера токена (JaCarta) если консольная утилита отсутствует. Реализована загрузка данных о пользователях из LDAP. Значения логина автоматически подставляются в поле "label" токена.
Приложение сокращает время на подготовку токена за счет использования одного интерфейса для токенов разных производителей, использования предустановленных настроек и загрузки данных из LDAP.
